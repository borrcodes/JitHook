ReadMe for Inline IL tool

by Mike Stall (http://blogs.msdn.com/jmstall )

Feb 22, 2005

 

See http://blogs.msdn.com/jmstall/archive/2005/02/21/377806.aspx for more details about the InlineIl tool and for an explanation of how it works.

 

Includes demo files:

  • Simple_CSharp.cs
  • Simple_VB.vb
  • Filter_CSharp.cs

 

 

Compiling

I’ve attached the source (program.cs) and compiled output (inlineil.cs).

Compile it via:

csc /out:inlineil.exe program.cs

 

Running

You can then run it via providing a file as a single argument, like so:

inlineil.exe Simple_CSharp.cs

It will determine the language based off extension (VB or C#). It only supports a single filename and not multi file projects.

 

InlineIL needs access to the SDK (to get Ildasm and peverify). It finds the sdk via the %FrameworkSDKDir% environment var, which is what a default command window from VS2005 will set.
